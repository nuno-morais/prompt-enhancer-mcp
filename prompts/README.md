# Reusable Prompt Templates

This directory contains reusable prompt templates for the Prompt Enhancer MCP project.

You can use these templates for:
- Standardizing the system prompts you feed to the local Ollama model.
- Providing consistent context for different "personas" (e.g. brainstorm mode vs standard mode).
- Having a library of quick copy-paste examples.

## Available Templates
- `system_prompt.md` - The base system prompt for the prompt engineer persona.
